Data visualization

Data visualzation is placing data in a visual context for easy understanding.

Data visualization is used to identify relationships and patterns.

D3.js what is it?

D3.js is a JavaScript library for manipulating DOM objects based on data.

D3.js is used for data visualization.

How is D3.js different?

Provides domain specific language for transforming the DOM.

D3 handles the rendering of the data, SVG, and CSS.

Syntax is similar to JQuery.

How to select items

Uses the W3C selectors to select items in the DOM.

To make a selection the global namespace d3 must be used.
